# CookMai

## 1. Environment Setup
Install the required dependencies using the following command:

```bash
pip install -r requirements.txt
```

## 2. Data and Model Preparation
Run the Jupyter notebooks in the notebooks folder in the following order to generate the .pkl files:
1. setup.ipynb
2. search_engine.ipynb
3. spell-collection.ipynb
4. image-collection.ipynb
5. recommendation_engine.ipynb

Note: Step 5 (recommendation_engine.ipynb) may take around 108 minutes or more than 2 hours to process.

## 3. Running the Project
Once the files are prepared, run the following command:

```bash
python app.py
```
